¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear páginas web. Es el esqueleto de un sitio web y se utiliza para estructurar contenido y documentos en la web.

Elementos Básicos de HTML

  • <html>: Define un documento HTML.
  • <head>: Contiene metadatos como título y enlaces a hojas de estilo.
  • <body>: Contiene el contenido visible de la página.
  • <title>: Establece el título de la página que aparece en la pestaña del navegador.

Estructura Básica de una Página HTML

<!DOCTYPE html>
<html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Mi Página Web</title>
    </head>
    <body>
        <h1>Bienvenidos a mi página web</h1>
        <p>Este es un párrafo de texto</p>
    </body>
</html>

Etiquetas Comunes en HTML

  • <a>: Define un enlace a otra página o recurso.
  • <img>: Inserta una imagen en la página.
  • <p>: Define un párrafo.
  • <h1>, <h2>, <h3>: Encabezados de diferentes niveles.
  • <ul> y <ol>: Listas desordenadas y ordenadas.
  • <table>: Crea tablas con filas y columnas.

Listas

Lista Desordenada

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>
  • Elemento 1
  • Elemento 2
  • Elemento 3

Lista Ordenada

<ol>
    <li>Primer paso</li>
    <li>Segundo paso</li>
    <li>Tercer paso</li>
</ol>
  1. Primer paso
  2. Segundo paso
  3. Tercer paso

Tablas

<table>
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
    <tr>
        <td>Juan</td>
        <td>30</td>
    </tr>
</table>
Nombre Edad
Juan 30

Formulario HTML

<form>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">
    <input type="submit" value="Enviar">
</form>

Incorporar CSS (Hoja de Estilos)

<style>
    body {
        font-family: 'Lora', serif;
        background-color: #f0f0f0;
    }
    h1 {
        color: #2c3e50;
    }
</style>
