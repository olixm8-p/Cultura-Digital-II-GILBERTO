¿Qué es HTML?

HTML significa "HyperText Markup Language" (Lenguaje de Marcado de Hipertexto). Es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

Historia de HTML

Fue creado en 1991 por Tim Berners-Lee, un físico del CERN. Desde entonces ha evolucionado considerablemente con nuevas versiones.

Versiones de HTML

  • HTML 1.0 (1991)
  • HTML 2.0 (1995)
  • HTML 3.2 (1997)
  • HTML 4.01 (1999)
  • XHTML (2000)
  • HTML5 (2014)

Elementos Básicos

<html>
    <head>
        <title>Título</title>
    </head>
    <body>
        <h1>Encabezado</h1>
        <p>Un párrafo.</p>
    </body>
</html>
